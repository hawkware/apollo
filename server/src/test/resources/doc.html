Resource URI Methods Description
Application http://localhost:8080/apollo/application/{applicationName} GET, POST, DELETE This resource contains application properties
Application collection http://localhost:8080/apollo/applications GET This resource represents the collection of applications
Context http://localhost:8080/apollo/context/{contextName} GET, POST, DELETE This resource contains Servers which are available in an application context
Context collection http://localhost:8080/apollo/contexts GETE This resource represents a collection of contexts
